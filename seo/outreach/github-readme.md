# Meccha Chameleon Art — Browser Fan Game

> A fan-made browser adaptation of the Steam game **Meccha Chameleon** by LEMORION.
> No download · 5 maps · 50 hiding spots · Hide-and-seek with paint-based camouflage.

**Live site**: https://mecchachameleon.art/

## What is this?

This is a community-run fan site for [Meccha Chameleon on Steam](https://store.steampowered.com/app/4704690/).
The Steam game is a multiplayer hide-and-seek game where hiders paint themselves to blend into the environment.
This browser version lets you practice camo strategies and explore 5 hand-picked maps with 50 hiding spot breakdowns — no install required.

## Features

- 🎨 50 hiding spots across 5 maps (5×2 grid view, all thumbnails on home page)
- 🗺️ Per-map pages with color analysis, difficulty rating, and tips
- 🌐 Bilingual (English + 中文)
- 📱 Mobile-friendly
- ⚡ No download, no signup, browser-only
