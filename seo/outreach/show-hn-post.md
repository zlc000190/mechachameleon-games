Title: Show HN: Meccha Chameleon Art – browser fan companion, 50 hide spots across 5 maps

Body:

Hi HN,

I built https://mecchachameleon.art/ — a fan-made browser companion for the Steam hide-and-seek game Meccha Chameleon (by LEMORION).

The Steam game is a multiplayer hide-and-seek where hiders paint themselves to blend in. I kept alt-tabbing to check screenshots of hiding spots, so I scraped the official screenshots, organized them by map, and built a static Next.js site that shows all 50 spots in a 5×2 grid with color analysis and difficulty ratings.

Tech: Next.js 14 (App Router) + ShipAny template, fully static export, mobile-first, bilingual (EN/中文).

Things I'd love feedback on:
- The screenshot-to-color-paint analysis — currently manual, want to automate
- Steam Workshop integration so community maps can flow in
- Better seeker-path prediction (which spots get found first?)

The site is fan-made, unofficial, and clearly disclaimed. Meccha Chameleon is © LEMORION.
